1. js是干嘛用的
   js 是对用户在进行页面操作是所做出的行为；
   （html 是页面的内容；
     css  是页面的布局；
     js   是页面的行为；）
2. 引入js文件
   script  是脚本的意思；
   如果是把js写在一个页面，那直接在head 或 body 里面写入 script即可，两个位置的效果都是一样的，只是浏览器的解析顺序不一样罢了；
   如果是外部文件的话，那么就在head 或 body 里面写入 script src 即可；
3. 
使用 window.alert() 弹出警告框。
   window.alert() 弹出警告框，是显示数据的方式之一，弹出警告框在屏幕的中上方；还需多加练习；
使用 console.log() 写入到浏览器的控制台。
   console是进行测试用的，为了减少bug,不断的进行测试；
   
4. js是一种轻量级的，弱类型的语言
    比如c语言可以定义 int， long， double， char
    但是在js里面只有var类型，它会根据你给的value值自动识别

    js是一种脚本语言，是一种轻量级的编程语言；
    js 里面只有var 类型，这里可以定义数组，字符串，等等；
5. 学会如下类型
var x;               // x 为 undefined
var x = 5;           // 现在 x 为数字
var x = "John";      // 现在 x 为字符
var cars=["Saab","Volvo","BMW"]; //数组
var person={firstname:"John", lastname:"Doe", id:5566}; //对象，其实就是json对象
